# CS3031-group5
## The Study Assistant Project 

This project uses the Adafruit Circuit Playground Express (CPX), along with several sensors and actuators, to create a study tool.

## Installation
- Wire up the stuff according to the diagram:
<img width=50% align=center alt="image" src="https://github.com/user-attachments/assets/f3a9e6a1-5fe4-4b89-ae4c-bb8c1b146f48" />

*Add lib files*

### Step 0
`git clone https://github.com/elka-blitz/cs3031-group5/`

`cd cs3031-group5`

### Step 1

Move the python files to the CPX

Move the `.mpy` binaries to the CPX `lib` folder
